Présentation

Bonjour, je m'appelle Alexis Gonçalves. Je suis actuellement en licence professionnelle : développement d'application intranet/internet à l'IUT de Clermont-Ferrand. Je travaille également en alternance dans le service informatique de l'URSSAF Auvergne. Mon objectif est de travailler, après ma licence, dans une entreprise de développement web dans laquelle je pourrais apporter mon expérience acquise lors de mon parcours universitaire, mais également enrichir mes compétences.

Mes principales compétences

HTML & CSS

C'est avec ces deux langages qu'a débuté mon intérêt pour l'informatique. J'ai commencé à les étudier, en autodidacte, lorsque j'étais en première STI2D en réalisant des petits sites web statiques.

JS

Durant mon parcours universitaire, j'ai également appris à me servir du langage JavaScript. Ce langage m'a tout de suite plus, car je le trouve intuitif.

PHP

Le PHP est le premier langage qui m'a fait découvrir le développement sur serveur. J'ai réalisé un parser de news lors de ma deuxième année de DUT. Ce projet m'a beaucoup plu.

Détermination

Dès le début de mon DUT, j'ai rencontré des difficultés de compréhension. Je n'ai cependant pas abandonné et ai participé à des tutorats (des services qui permettent aux étudiants de première année de se faire aider par des étudiants de deuxième année et de licence) durant cette période. Grâce aux différentes aides que j'ai eues, j'ai pu surmonter mes obstacles.

Autodidacte

Pour minimiser les difficultés rencontrées durant mon parcours universitaires, j'ai pris l'initiative de me renseigner à l'avance sur les matières que je verrais. Par la suite, j'ai suivi quelques cours sur openclassrooms.com afin d'avoir un aperçut général des différents langages que je verrai.

Je possède d'autres compétences. N'hésitez pas à consulter cette page afin d'en prendre connaissance.

Réalisation

Découverte HTML

Le premier site web que j'ai réalisé (durant mon baccalauréat STI2D) m'a surtout servi à acquérir les bases de l'HTML et du CSS. Il est composé d'un tableau qui reprend quelques pokémons avec diverses caractéristiques.

Utilisation d'une API pour faire une liste de pokémon

Pour apprendre à me servir d'une API et de JavaScript, j'ai réalisé un script qui réalise des requêtes à une API pokémons et affiche ces derniers sur une page web.

Réalisation d'une PWA (Progressive Web App)

Ce qui m'a dernièrement intéressé est les PWA, des sites web qui peuvent s'installer à l'aide d'un navigateur récent. J'ai pour cela suivi un tutoriel d'une chaîne YouTube. Grâce à ce tutoriel, j'ai réalisé une simple PWA qui permet d'ajouter et de supprimer des livres. Ces derniers sont stockés sur le service proposé par Google : Firestore.

Réalisation d'une page pour présenter mes compétences

Afin de mettre en avant mes compétences, j'ai décidé de réaliser une page qui les listerait. Sur cette page, j'ai noté chacune de mes compétences avec une note sur 10. Plus la note d'une compétence est élevée, plus je me sens à l'aise avec celle-ci. Mon objectif est de maintenir cette page à jour en fonction de mon évolution. J'ai utilisé le framework VueJS pour la mise en place de l'interface. J'ai également développé une API REST sur Heroku qui interagit avec une base de données PostgreSQL pour le stockage et la manipulation de mes compétences.
